# Aisland - Dynamic Island for Mac

A macOS utility that transforms the MacBook notch into an interactive widget dashboard.

## Technology Stack
- Swift 5.x
- SwiftUI + AppKit
- SwiftData for persistence
- Minimum macOS: 14.6
